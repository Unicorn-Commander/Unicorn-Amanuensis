KEY DIFFERENCES:
----------------

1. CHECKBOX BEHAVIOR CHANGE:
   Before: Diarization was CHECKED by default
   After:  VAD is CHECKED, Diarization is UNCHECKED

2. NEW VAD CONTROL:
   - Allows users to disable silence filtering if needed
   - Checked by default (most users want this)
   - Sends vad_filter=true/false to server

3. IMPROVED DIARIZATION CONTROL:
   - Now unchecked by default (experimental feature)
   - Better labeling with warning about experimental status
   - Sends enable_diarization=true/false to server

4. VISUAL FEEDBACK:
   - Results now show what settings were used
   - Green "âœ“ ON" for enabled features
   - Muted "âœ— OFF" for disabled features

5. BETTER UX:
   - Larger checkboxes (18px vs default)
   - Descriptive text under each option
   - Pointer cursor on hover
   - Consistent spacing and alignment
